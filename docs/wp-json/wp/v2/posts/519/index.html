First Contact<\/p><\/div>\n

Date: Thursday 26th<\/sup> September 2013<\/strong><\/p>\n

Time: 7:00pm<\/strong><\/p>\n

<\/strong>Venue: Staffordshire University – Stafford Campus<\/a> (K106 – the Octagon building)<\/strong><\/p>\n

 <\/p>\n

Last month we continued our discussions and began creating the RESTful implementation of our alien invasion game with the creation of the game controller.<\/p>\n

Continuing with getting the minimal feature set completed, this month we look to address the missing features to move us towards our first RESTful client integration. \u00a0The server needs to respond to requests for a wave and generate responses to a defence strategy. Once the simple implementation of London is working, we can then look towards multiple stages, persistent weapons states and teams.<\/p>\n
